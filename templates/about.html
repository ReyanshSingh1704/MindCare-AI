About MindCare AI

A simple Mental Health Support Chatbot made using Flask and Python.

📖 About This Project

MindCare AI is a simple chatbot that allows users to share their feelings. It analyzes the user's message, detects emotions and risk level, and gives supportive responses.

It also stores chat history and displays analytics using charts.


✨ Features

  • 💬 AI Chatbot
  • 😊 Emotion Detection
  • ⚠️ Risk Detection
  • 📊 Dashboard with Charts
  • 📜 Chat History
  • 🗑 Delete & Clear Chats
  • 📱 Responsive Design

🛠 Technologies Used

  • Python
  • Flask
  • SQLite
  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • TextBlob
  • Pandas
  • Matplotlib

🎯 Project Objective

The main goal of this project is to create a simple web application that helps users express their emotions while learning Flask, databases, sentiment analysis, and web development.
